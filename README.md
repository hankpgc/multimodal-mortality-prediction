# 深度多路徑神經網路於 ILD/COPD 患者死亡風險預測

> **碩士論文** · 國立中興大學 資訊工程學系 · 2025  
> 指導教授：吳俊霖 · 研究生：彭桂綺

---

## Description

本研究針對**間質性肺病（ILD）**與**慢性阻塞性肺病（COPD）**患者，提出一套可預測一年內死亡風險的深度多路徑神經網路框架。

傳統臨床評估工具（如 BODE、GAP 指數）僅依賴少數肺功能指標，對早期無症狀患者判別效能有限。本研究整合**胸部 CT 影像**與 **94 項臨床檢測指標**，設計具缺失模態容錯能力的多模態融合模型：

- 五折交叉驗證 AUC **0.872**，顯著優於單模態基準（p < 0.05）
- 結合 **SHAP** 與 **Grad-CAM** 提供可解釋性分析，協助臨床醫師理解預測依據
- 支援資料不完整樣本（缺失模態容錯），排除後仍維持 AUC **0.853**

## Results

| 方法 | AUC (95% CI) | F1-Score | Accuracy |
|------|-------------|----------|----------|
| 影像單模態（CT only） | 0.758 (0.725–0.791) | 0.775 | 0.769 |
| 臨床單模態（Clinical only） | 0.840 (0.775–0.905) | 0.838 | 0.833 |

| **本研究（雙模態融合）** | **0.872 (0.798–0.946)** | **0.861** | **0.867** |

## Implementation

### Dataset

- **來源**：台中榮民總醫院 2022 年 ILD / COPD 患者，共 72 例
- **影像模態**：每位患者擷取 6 張 axial CT 切片，轉為灰階並進行強度正規化
- **臨床模態**：94 項變數，涵蓋心律貼片（35 項）、CPET（16 項）、肺功能、睡眠品質等；缺失值以中位數填補，再進行 Min-Max 正規化

### Model Architecture

```
CT Image (6 slices)
    └─ DenseNet-121 (2D, 1-channel)
    └─ Slice Attention
    └─ Linear → dim × 16  →─┐
                              ├─ Fusion Net (LeakyReLU) → P3 (死亡機率)
Clinical Data (94 vars)       │
    └─ MLP (94→64→32→16)  →─┘
    └─ Linear → dim × 16
```

- **影像分支**：預訓練 DenseNet-121（修改為單通道輸入），接 Slice Attention 對 6 張切片加權融合，輸出 16 維向量
- **臨床分支**：MLP 加入 Batch Normalization 與 Dropout（0.3）防止過擬合，輸出 16 維向量
- **Fusion Net**：兩分支向量串接後，透過全連接層與 LeakyReLU 輸出死亡機率
- **缺失模態容錯**：支援「僅影像」、「僅臨床」、「雙模態」三種輸入情境，各路徑獨立計算 BCEWithLogitsLoss，加權合併為總損失

### Training

- **框架**：PyTorch 2.4.1，CUDA 12.0
- **硬體**：NVIDIA TITAN RTX（24 GB VRAM），Ubuntu 22.04
- **驗證策略**：五折交叉驗證，採梯度裁剪（gradient clipping）穩定訓練

### Explainability

- **SHAP**：識別關鍵臨床危險因子，主要為低血氧飽和度（CPE7）、血流動力學下降（CPE4）、高死腔換氣比（CP07）
- **Grad-CAM**：視覺化影像關注區域，熱區集中於肺部纖維化與蜂窩化病灶，符合臨床判斷
